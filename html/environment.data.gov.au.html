environment.data.gov.au, tombstoned

The domain environment.data.gov.au was once a Persistent Identifier (PID) domain that was used to create PIDs for several CSIRO projects in 2018 - 2022.

This domain is no longer used and all PIDs associated with it are no longer expected to resolve.

If you are seeing this page, you might have clicked on a environment.data.gov.au PID link in data or a web page, and you've come here.

If you wish to find/create/understand Australian Government PIDs, like those made for environment.data.gov.au, please visit the Australian Government Linked Data Working Group's website:

  • https://www.linked.data.gov.au
