Non-resolving PID

The Australian Government Linked Data Working Group Persistent Identifier (PID) web address that you resolved does not resolve!

The PID remains known to the AGLDWG and may be in use in data identifying something, but it does not resolve to a publicly available web resource.

There are several reasons as to why this may have occured:

  • The system delivering public data about the resource is dead
  • The resource has no public information. It may have private (non-public) information that only resolves on a closed network
  • The PID's delivering system is not yet functional

For more information, please contact the PID owner. If you don't know who that is, brows the AGLDWG's catalogue fo PIDs and look for the PID namespace to find the details. If you are seeing this page, there must be an entry in that cataloguf for the PID's namespace.
